# CRM Module Demo

Module de suivi commercial interne — gestion de clients et d'opportunités de vente.

**Stack :** NestJS · Prisma · PostgreSQL · Next.js App Router · TypeScript

---

## Lancement en moins de 5 minutes

### Prérequis
- [Docker](https://www.docker.com/products/docker-desktop) installé et lancé

### Démarrage

```bash
git clone https://github.com/maagliyosri/crm-module-demo.git
cd crm-module-demo
docker-compose up --build
```

L'application est disponible sur :
- **Frontend** → http://localhost:3000
- **Backend API** → http://localhost:3001
